# mxzou.com

Minimal portal linking two identities:

- [mengxuanzou.com](https://mengxuanzou.com) — cinematic sound
- [stussysenik.com](https://stussysenik.com) — hip-hop engineering

## Stack

- SvelteKit
- DM Sans + Inter fonts
- Blue ink on paper white

## Development

```bash
bun install
bun run dev
```

## Build

```bash
bun run build
```
